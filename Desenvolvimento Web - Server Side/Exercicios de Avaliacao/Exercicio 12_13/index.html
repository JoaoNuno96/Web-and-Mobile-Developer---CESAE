Exercicio 12,13

No HTML crie uma lista intitulada “O que já aprendi em JS:” que contenha os seguintes itens: Lógica de Código JS, Funções, DOM, Eventos, e um botão “aprendido”. Ao clicar no botão, em JS adicione no final de todos os itens “– aprendido”.

Crie um botão em html que lance um prompt ao utilizador a perguntar uma outra coisa que tenha aprendido. Em JS, adicione o item à lista anterior.
